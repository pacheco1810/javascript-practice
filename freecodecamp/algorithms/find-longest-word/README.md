# 🧠 Find Longest Word Length

This exercise implements different approaches to find the **length of the longest word in a given sentence**.

---

# 📌 Problem

Given a string, return the **length of the longest word**.

A word is defined as a sequence of characters separated by spaces.

---

## 🧪 Example

### Input

```
"May the force be with you"
```

### Output

```
5
```

### Explanation

| Word  | Length |
| ----- | ------ |
| May   | 3      |
| the   | 3      |
| force | 5      |
| be    | 2      |
| with  | 4      |
| you   | 3      |

👉 The longest word is **"force"**, with a length of **5**.
