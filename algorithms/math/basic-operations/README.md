# Basic Operations

This exercise implements basic arithmetic operations using arrow functions.

---

# Problem

Create functions for basic arithmetic operations:

* Addition (add)
* Subtraction (rest)
* Multiplication (mul)
* Division (div)

Each function takes two parameters and returns the result.

---

### Example

Input:

div(10, 5)

Output:

2

# Solutions Implemented

---

## 1. add

## 2. rest
