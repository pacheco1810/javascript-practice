# Algorithms Practice

This folder contains algorithm and problem-solving exercises used to strengthen my programming fundamentals in JavaScript.

The goal of these exercises is to improve logical thinking, data manipulation skills, and preparation for technical interviews.

## Topics

The algorithms are organized by topic:

* **arrays** – problems involving array manipulation, iteration and indexing
* **strings** – string processing and text manipulation
* **math** – numerical algorithms and mathematical problems
* **recursion** – recursive problem solving

## Purpose

These exercises help me practice:

* Algorithmic thinking
* Time and space complexity awareness
* Writing clean and readable JavaScript
* Solving programming challenges efficiently

## Example Structure

```
algorithms
│
├── arrays
│   └── two-sum.js
│
├── strings
│   └── reverse-string.js
│
├── math
│   └── fibonacci.js
│
└── recursion
    └── factorial.js
```

Each file contains a single algorithm implementation and sometimes additional comments explaining the solution.
