# Filter Even Numbers

This exercise implements different approaches to filter even numbers from an array.

---

# Problem

Given an array of numbers, return a new array containing only the even numbers.

---

### Example

Input:

[1, 2, 3, 4, 5, 6]

Output:

[2, 4, 6]

# Solutions Implemented

---

## 1. Classic Loop (Algorithmic Approach)

* Iterates through the array manually
* Checks if each number is even
* Pushes valid numbers into a new array

Purpose:
Practice control flow and manual array building.

---

## 2. forEach Loop

* Iterates using `forEach`
* Applies condition inside callback
* Builds result array

Purpose:
Practice array iteration and conditional logic.

---

## 3. Functional Approach (filter)

* Uses `filter`
* Returns elements that satisfy a condition

Purpose:
Practice higher-order functions and functional programming.

---
